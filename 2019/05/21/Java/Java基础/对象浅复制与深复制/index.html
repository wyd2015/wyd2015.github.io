Java对象的深/浅复制

结论： 如果要做到深拷贝，引用类型变量所指向的类必须：

  • 实现Cloneable接口，并重写clone()方法；
  • 重写clone()方法时，将clone()方法的修饰限定符由protected改为public。

浅拷贝：拷贝对象时，基本类型变量会重新复制一份，而引用类型的变量只是对引用进行了拷贝。并没用拷贝引用变量所指向的实际对象。

深拷贝：拷贝对象时，把基本类型变量和引用变量所指向的实际对象都重新拷贝一份。
