这里需要注意一个问题：

springboot默认只识别（扫描）java/路径下的 *.java 文件，也即默认情况下无法扫描到位于的 src/main/java/ 路径下的 *Mapper.xml文件。正确的方式是将 *Mapper.xml 映射文件放到 resource/ 路径下。

上述错误就是因为找不到 *Mapper.xml 文件的位置导致的！
